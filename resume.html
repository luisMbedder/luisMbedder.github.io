Personal Projects

SpectrumScan

SpectrumScan is a homebrew 2GHz Spectrum Analyzer built on a Beaglebone single-board computer using a USB RTL-SDR(Software Defined Radio) reciever to capture and display the radio frequencies all around us. The software is written in C++ and uses a 7.0" touchscreen display for the GUI built using the Qt framework. Development was done on an Ubuntu x86 machine and cross-compiled for the ARM beaglebone target running debian linux. The project is hosted on github and released under the GPL v.3 License. For demo and sources visit the SpectrumScan project page, http://luismbedder.com/SpectrumScan
